This Tab Will Self Destruct™

Tabs will close after of inactivity.

Tabs won't self destruct if they're pinned or currently active.

Defuse a self destructing tab by clicking the icon on the toolbar and choosing Defuse.

This is an open source program. The source code is available on GitHub. Please feel free to contribute or modify for your own purposes. If you have a bug report or, better yet, know how to fix one, that should be done at GitHub too.
